# SmartDine-AI
SmartDine AI: Platformă de management restaurant cu acces via QR Code. Clienții comandă fără cont, asistați de un agent AI pentru recomandări personalizate. Include un dashboard securizat pentru staff (Bucătari/Ospătari) cu update-uri în timp real prin WebSockets. Eficiență, viteză și experiență digitală modernă.
